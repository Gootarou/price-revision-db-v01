# 料金改定計算DB 改良版 v0.1

## プロジェクト概要

本プロジェクトは、Googleスプレッドシート + Google Apps Script で構築する、料金改定計算・確認DBです。

v1は文書生成システムではありません。v1は、料金改定に必要な案件情報、勤務時間根拠、最低賃金、計算係数を管理し、計算結果を確認するためのDBです。Word等への文書転記は、人間が確認ビューを見ながら手動で行います。

## 主要機能

* 初期セットアップ
* 入力画面初期化
* 案件保存
* 案件読込
* 保存して計算
* 勤務時間根拠確認サイドバー
* 文書転記前確認ビュー更新

## 主要シート

| シート名 | 役割 |
| --- | --- |
| 入力_案件編集 | 新規登録、既存案件読込、保存、計算の入口となる入力画面。 |
| データ_料金改定案件 | 年度ごとの料金改定案件、料金条件、計算結果、確認状態を保持する主データ。 |
| 明細_勤務時間根拠 | 案件に紐づく勤務時間根拠の子明細を保持する主データ。 |
| 確認_文書転記前 | Word等へ手動転記する前に、必要項目を一覧確認するための参照ビュー。 |
| マスタ_契約サービス | 契約サービスID、契約先、現場、業務名などの契約サービス基本情報。 |
| マスタ_契約先 | 契約先名、宛名、住所などの契約先情報。 |
| マスタ_現場 | 現場名、契約先ID、都道府県などの現場情報。 |
| マスタ_コード対応表 | 契約先、現場、テナント、物件コードなどの対応関係。 |
| マスタ_最低賃金 | 都道府県別の最低賃金と適用開始日。 |
| マスタ_計算係数 | 法定福利費計算倍率など、計算に使う係数。 |
| 管理_設定 | 採番やシステム設定値を保持する管理用シート。 |
| 管理_処理ログ | GAS処理の実行ログを保持する管理用シート。 |

## 正本の考え方

* 入力_案件編集 は正本ではありません。
* 確認_文書転記前 は正本ではありません。
* 勤務時間根拠確認サイドバーは正本ではありません。
* 正本は データ_料金改定案件 と 明細_勤務時間根拠 です。
* マスタ類は参照用の基礎データです。
