# 運用メモ

## 基本運用フロー

1. マスタ類を整備する
2. 入力_案件編集 を初期化する
3. 案件情報を入力する
4. 勤務時間根拠を入力する
5. 保存する
6. 保存して計算する
7. 必要に応じて勤務時間根拠サイドバーで確認する
8. 文書転記前確認を更新する
9. 確認_文書転記前 を見ながらWord等へ手動転記する

## 注意事項

* 入力_案件編集 は正本ではない。
* 確認_文書転記前 は正本ではない。
* サイドバーは確認専用であり、編集・保存・計算実行には使わない。
* 計算対象は月額のみ。
* 月額以外は対象外。
* 最低賃金参照年月は人間が入力しない。
* 最低賃金は日付から自動参照する。
* 勤務時間根拠の詳細は 明細_勤務時間根拠 に保存する。
* データ_料金改定案件 に勤務時間根拠の詳細列を増やさない。
