## v0.1の正本ルール

* 入力_案件編集 は正本ではありません。
* 確認_文書転記前 は正本ではありません。
* 勤務時間根拠確認サイドバーは正本ではありません。
* 正本は データ_料金改定案件 と 明細_勤務時間根拠 です。
* マスタ類は参照用の基礎データです。

## v0.1実運用テスト前の確認

* PRがすべてmainにマージ済みであること。
* ローカルで `git pull origin main` 済みであること。
* `apps-script` から `clasp push` 済みであること。
* 対象スプレッドシートが `.clasp.json` の `scriptId` に紐づくものであること。
* スプレッドシート側でメニューが表示されること。
* `tests/manual-test-checklist.md` に沿って手動テストを行うこと。
