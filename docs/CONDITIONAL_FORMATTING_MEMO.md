# 条件付き書式メモ

## 方針

* GASでは値や状態に応じた色変更をしない。
* 条件付き書式は人間がスプレッドシート側で設定する。
* GASで条件付き書式の自動生成はしない。

## 候補

人間がスプレッドシート側で設定する条件付き書式の候補は以下の通り。

* 計算結果状態 = エラー
* 計算結果状態 = 対象外
* 計算結果状態 = OK
* 確認状態 = 未確認
* 確認状態 = 確認済
* 文書転記OK = OK
* 文書転記OK = NG
* 交渉状況 = 承認
* 交渉状況 = 見送り
* エラー内容 が空欄ではない

このPhaseでは条件付き書式を実装しない。あくまで人間が設定するためのメモとする。
